# Sistema web de E-commerce
Api voltado para gerenciar compras online.

## execução
executar applicação:
`./mvnw spring-boot:run`

## rotas
- /categories - GET
    - /id - GET
      
- /orders - GET
    - /id - GET
      
- /products - GET
    - /id - GET

- /users - GET
    - /id - GET, POST, DELETE, PUT
